### 时间倒数日

## 创建原因


- 因为宿舍作息不一致，导致我的睡眠很不好，只能期盼每个月回一次家，改善睡眠，面对度日如年的时间，于是我想要做一个可以给我看倒数日的项目
- 其实这个需求还是挺多的，像高考的倒计时，考公的倒计时，纪念日等等
- 于是我就产生了做这个项目的想法

## 技术栈
- Vue


## 目前计划
- 实现倒数日功能
- 有用户登录页面，每个用户的存储可保留
- 有用户个性化定制服务
